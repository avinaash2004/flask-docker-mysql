# Flask + MySQL with Docker Compose

This project is a simple Flask application that connects to a MySQL database using Docker Compose.

## Features

- Flask web server
- MySQL database
- Docker + Docker Compose setup

## How to Run

```bash
docker-compose build
docker-compose up
